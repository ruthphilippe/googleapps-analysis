# Google Play Store Apps Analysis

This project uses Python in Google Colab to clean and analyze a Google Play Store apps dataset.

## Project goals
- Clean messy columns such as Installs, Price, Size, and Reviews
- Handle missing values
- Explore app categories, ratings, installs, and pricing

## Tools used
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Key findings
- Family has the most apps
- Events has the highest average rating
- Game has the most installs
- Free and paid apps have similar average ratings
- Most paid apps are priced between $0 and $50
